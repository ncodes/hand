# Troubleshooting

This page should collect recurring setup and runtime problems.

## Content Outline

- Daemon unreachable.
- Provider auth missing or invalid.
- SQLite FTS5 build tags.
- Gateway receives no Slack/Telegram events.
- Webhook 401/404 issues.
- Memory and vector search warnings.
- TUI rendering or terminal issues.
