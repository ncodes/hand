# Gateway Overview

Hand's gateway runs inside the daemon and connects external clients to the same agent runtime.

## Content Outline

- Gateway lifecycle and `hand gateway status/start/stop/restart`.
- Generic HTTP, Slack, and Telegram modes.
- Conversation binding and session continuity.
- Sender allowlists and pairing.
- Local-first defaults: Slack Socket Mode and Telegram polling.
- Hosted options: Slack Events API and Telegram webhooks.
