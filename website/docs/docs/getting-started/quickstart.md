# Quickstart

This page should walk a new user from repository checkout to a working Hand conversation.

## Content Outline

- Requirements: Go version, provider credentials, and local build assumptions.
- Build or run Hand from source.
- Initialize or select a profile.
- Store provider credentials with `hand auth login`.
- Start `hand daemon start`.
- Open the TUI or send a one-shot prompt.
- Run `hand doctor` to verify readiness.
- Link to profiles, model auth, TUI, and troubleshooting.
